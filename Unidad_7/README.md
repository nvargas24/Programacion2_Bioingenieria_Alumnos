# Unidad 7: Contadores y Temporizadores

## Descripción

En esta unidad se estudia el uso de **timers y contadores**, periféricos fundamentales en microcontroladores para el manejo del tiempo.

## Objetivos

- Comprender el funcionamiento de temporizadores
- Generar eventos periódicos
- Medir intervalos de tiempo

## Contenidos

- Temporizadores
- Contadores
- Prescalers
- Generación de interrupciones por timer

## Estructura de los ejemplos

Los ejemplos incluyen:

- Generación de delays
- Blink LED con timer
- Medición de tiempo
