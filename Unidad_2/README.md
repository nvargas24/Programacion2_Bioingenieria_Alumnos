# Unidad 2: Entornos Gráficos

## Descripción

En esta unidad se introduce el desarrollo de **interfaces gráficas de usuario (GUI)** utilizando el framework **Qt**.  
Se presentan los conceptos básicos de **widgets, layouts y programación orientada a eventos**.

## Objetivos

- Comprender el funcionamiento de una interfaz gráfica
- Utilizar Qt Designer para crear interfaces
- Conectar eventos de interfaz con código
- Desarrollar aplicaciones simples con GUI

## Contenidos

- Introducción a Qt
- Widgets básicos
- Layouts
- Programación orientada a eventos
- Señales y slots

## Estructura de los ejemplos

Los ejemplos incluyen:

- Creación de interfaces con Qt Designer
- Uso de widgets básicos
- Manejo de eventos
- Aplicaciones simples con botones y campos de texto

## Requisitos

- Qt Creator
- Compilador C++
