# Unidad 5: Introducción a GPIO

## Descripción

En esta unidad se introduce el uso de **GPIO (General Purpose Input/Output)** para interactuar con el mundo físico mediante pines digitales.

## Objetivos

- Configurar pines como entrada o salida
- Leer señales digitales
- Controlar dispositivos simples como LEDs

## Contenidos

- Concepto de GPIO
- Configuración de pines
- Entradas digitales
- Salidas digitales

## Estructura de los ejemplos

Los ejemplos incluyen:

- Programa "Blink LED"
- Lectura de pulsadores
- Control básico de dispositivos digitales
