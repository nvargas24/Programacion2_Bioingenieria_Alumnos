# Unidad 9: Máquinas de Estado

## Descripción

En esta unidad se introduce el concepto de **máquinas de estado**, una técnica fundamental para estructurar programas en sistemas embebidos.

## Objetivos

- Comprender el concepto de estado
- Modelar sistemas mediante máquinas de estado
- Implementar máquinas de estado en código

## Contenidos

- Estados
- Transiciones
- Diagramas de estado
- Implementación en software

## Estructura de los ejemplos

Los ejemplos incluyen:

- Control de semáforos
- Sistemas reactivos simples
