# Unidad 11: Introducción a la Medición y Generación de Señales Analógicas

## Descripción

Esta unidad introduce el manejo de **señales analógicas** en microcontroladores.

## Objetivos

- Comprender la conversión analógica-digital
- Medir señales analógicas
- Generar señales analógicas

## Contenidos

- Conversores ADC
- Conversores DAC
- Lectura de sensores analógicos
- Generación de señales

## Estructura de los ejemplos

Los ejemplos incluyen:

- Lectura de un potenciómetro
- Visualización de datos analógicos
