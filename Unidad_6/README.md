# Unidad 6: Polling e Interrupciones

## Descripción

En esta unidad se presentan dos técnicas fundamentales para el manejo de eventos en sistemas embebidos: **polling** e **interrupciones**.

## Objetivos

- Comprender el concepto de polling
- Comprender el funcionamiento de interrupciones
- Comparar ambos métodos

## Contenidos

- Polling
- Interrupciones
- Rutinas de servicio de interrupción (ISR)
- Ventajas y desventajas

## Estructura de los ejemplos

Los ejemplos incluyen:

- Lectura de botones mediante polling
- Manejo de eventos mediante interrupciones
