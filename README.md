# Programacion2 - Bioingenieria
Repositorio con ejemplos para la catedra de Programacion 2 - Bioingeniria  - UTNFRH

Repositorio de ejemplos utilizado en la cátedra para acompañar las prácticas de **programación en C++**, **entornos gráficos** y **programación de microcontroladores**.

El objetivo de este repositorio es proporcionar ejemplos simples y progresivos que permitan a los estudiantes comprender los conceptos fundamentales vistos en clase y experimentar con ellos.

Cada unidad contiene ejemplos de código, pequeños programas de prueba y demostraciones de los conceptos presentados durante la cursada.

---

# Organización del Repositorio

El repositorio está organizado por **unidades temáticas**.  
Cada carpeta contiene ejemplos de código relacionados con los conceptos vistos en clase.

## Estructura del repositorio
- 📦 **ejemplos-programacion-embebidos**
  - 📁 **Unidad_1_Cpp_Estructuras_Avanzadas** : Introducción a C++ y Programación Orientada a Objetos
  - 📁 **Unidad_2_Entornos_Graficos** : Desarrollo de interfaces gráficas con Qt
  - 📁 **Unidad_3_Microcontroladores_Sistemas_Embebidos** : Conceptos fundamentales de sistemas embebidos
  - 📁 **Unidad_4_Arquitectura_Microcontrolador**  : Registros y periféricos
  - 📁 **Unidad_5_GPIO**  : Configuración de pines digitales como entrada y salida
  - 📁 **Unidad_6_Polling_Interrupciones**  : Manejo de eventos mediante polling e interrupciones
  - 📁 **Unidad_7_Temporizadores**  : Uso de timers y contadores
  - 📁 **Unidad_8_GPIO_Avanzado**  : Configuraciones avanzadas de pines digitales
  - 📁 **Unidad_9_Maquinas_de_Estado**  : Modelado de sistemas mediante máquinas de estado
  - 📁 **Unidad_10_Comunicacion_Serie**  : Comunicación entre dispositivos mediante UART
  - 📁 **Unidad_11_Senales_Analogicas**  : Conversión analógica-digital y generación de señales
  - 📁 **Unidad_12_Metodos_Numericos**  : Aplicación de métodos numéricos básicos
---

# Requisitos

Para utilizar los ejemplos puede ser necesario contar con algunas de las siguientes herramientas:

* Compilador de C/C++
* Qt Creator
* Entorno de desarrollo para microcontroladores
* Placas de desarrollo utilizadas en la cátedra (LPC845)
