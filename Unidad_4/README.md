# Unidad 4: Introducción a la Arquitectura de un Microcontrolador

## Descripción

En esta unidad se estudia la **arquitectura interna de un microcontrolador**, analizando sus principales bloques funcionales.

## Objetivo

- Comprender la estructura interna de un microcontrolador

## Contenidos

- Registros
- Periféricos
- Buses internos

## Estructura de los ejemplos

Los ejemplos incluyen:

- Acceso a registros
- Lectura y escritura en memoria
- Configuración básica de periféricos
