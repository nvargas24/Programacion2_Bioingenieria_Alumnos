# Unidad 1: El Lenguaje C++ y las Estructuras Avanzadas de Datos

## Descripción

En esta unidad se introduce el lenguaje **C++** como evolución del lenguaje **C**, incorporando el paradigma de **programación orientada a objetos (POO)**.  
Se presentan los conceptos fundamentales de **clases, objetos, encapsulamiento y gestión de memoria**, que servirán como base para el desarrollo del resto de la materia.

## Objetivos

- Comprender las diferencias entre C y C++
- Introducir el paradigma de programación orientada a objetos
- Implementar clases y objetos
- Comprender el concepto de encapsulamiento
- Utilizar constructores y destructores
- Manejar memoria dinámica en C++

## Contenidos

- Evolución de C hacia C++
- Clases y objetos
- Miembros públicos y privados
- Encapsulamiento
- Constructores y destructores
- Entrada y salida con `iostream`
- Gestión dinámica de memoria (`new` y `delete`)
- Introducción a tipos abstractos de datos

## Estructura de los ejemplos

Los ejemplos de esta unidad muestran:

- Definición de clases
- Creación de objetos
- Uso de métodos
- Manejo de memoria dinámica
